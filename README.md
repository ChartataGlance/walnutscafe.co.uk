# Walnuts Cafe production website

Static mobile-first website for `walnutscafe.co.uk`.

## Deploy

1. Create GitHub repository: `chartataglance/walnutscafe.co.uk`
2. Upload all files from this folder to repository root.
3. In GitHub: Settings → Pages → Deploy from branch → `main` → `/ (root)`.
4. Set custom domain: `walnutscafe.co.uk`.
5. Enable Enforce HTTPS once GitHub completes certificate provisioning.
6. Add Cloudflare DNS records from `cloudflare-github-pages-rules.md`.

## SEO files included

- `robots.txt`
- `sitemap.xml`
- Canonical tags
- Open Graph tags
- Twitter card tags
- Organization, Restaurant, Menu and location schema
- `CNAME` for GitHub Pages custom domain
